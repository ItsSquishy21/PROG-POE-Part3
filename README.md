# CMCS - Claim Management System

CMCS is a web-based prototype for managing lecturer claims. It allows users to:

- Submit monthly claims with supporting documents.
- Track the status of submitted claims (Pending, Approved, Rejected).
- Download uploaded files.
- View claim details.

## Features

- ASP.NET Core MVC with Entity Framework Core
- SQL Server database backend
- File uploads (.pdf, .docx, .xlsx)
- Dark-themed responsive UI using Bootstrap 5
